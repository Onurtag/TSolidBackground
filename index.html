    • TSolidBackground

    AIO Autohotkey script that can make any window pseudo-fullscreen using padding over window borders and background.

    It can also move/resize windows and make them always on top.

    Project maintained by Onurtag Hosted on GitHub Pages — Theme by mattgraham

    TSolidBackground

    TSolidBackground allows you to immerse in a window completely by creating a solid background and hiding window borders. In a different saying, it tries to mimic fullscreen for any window without resizing it.

    At first I made TSolidBackground to stop distracting myself when playing VNs and later added more features that were useful for me.

    How to use TSolidBackground. (README)

    Default Hotkeys

    TSolidBackground hotkey: !T (! represents the Alt key. Read the hotkey guide below.)
Always On Top: !Y (Means Alt+Y and vice versa.)
Center window: !G
Show/Hide taskbar: !F
Advanced Options: !U
Suspend all other hotkeys: F8
Default Color: 250000 (hex value without the #)
    All hotkeys and many more options can be changed in the ini.

    More info is available in the README.
