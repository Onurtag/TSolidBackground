    TSolidBackground

    TSolidBackground allows you to immerse in a window completely by creating a solid background and hiding window borders.
    In a different saying, it tries to mimic fullscreen for any window without resizing it.

    At first I made TSolidBackground to stop distracting myself when playing Visual Novels and later added more features that were useful for me.

    Things To Know

    Default Hotkeys & Options (You can create a TSolidBackground.ini using advanced options):  
TSolidBackground hotkey: +T (meaning Shift+T)
Always On Top: +Y (meaning Shift+Y)
Center window: +G (meaning Shift+G)
Show/Hide taskbar: +F (meaning Shift+F)
Advanced Options: +U (meaning Shift+U)
Suspend all other hotkeys: F8
Default Color: 051523 (hex)
Custom rectangle width left: 0
Custom rectangle width right: 0
Custom rectangle height top: 0
Custom rectangle height bottom: 0
Enable Startup Window: 1
